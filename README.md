# İçerdenBilgi

> 🚧 **This project is currently under active development.**

İçerdenBilgi is a modern platform where users can anonymously share interview experiences and workplace insights to help others make informed career decisions. The goal is to make the hiring process more transparent by providing real experiences shared by candidates and employees.

---

## ✨ Tech Stack

| Layer                  | Technology                     |
| :--------------------- | :----------------------------- |
| **Frontend**           | Next.js 16 (App Router)        |
| **Language**           | TypeScript                     |
| **Styling**            | Tailwind CSS                   |
| **Data Fetching**      | TanStack Query                 |
| **Authentication**     | Firebase Authentication        |
| **Session Management** | Firebase Admin Session Cookies |
| **Backend**            | Express.js                     |
| **ORM**                | Prisma                         |
| **Database**           | PostgreSQL (Supabase)          |

---

## ✨ Current Features

- Anonymous experience sharing
- Company pages
- Google Authentication
- Email & Password Authentication
- Secure session-based authentication
- Protected API routes
- Responsive user interface
- SEO-friendly architecture powered by Next.js App Router
